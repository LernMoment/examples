Generate efficient source code from UML state diagrams and activity diagrams!

Sinelabore_RT_ was built especially for embedded software developers. It focuses on just one task: Generation of readable and maintainable code from hierarchical UML state machines and from activity diagrams. With its unique features the tool covers well the requirements of embedded real-time and low power application developers.

For more information see www.sinelabore.com
